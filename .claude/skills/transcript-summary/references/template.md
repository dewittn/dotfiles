## Conditional Elements

- **No event name known**: Omit that line, use topic as `#` header
- **No presenter stated**: Omit presenter line
- **No timecodes**: Omit `[MM:SS]` prefixes throughout  
- **No Q&A session**: Omit entire Q&A section
- **Fewer than 6 clear themes**: Use as many distinct themes as exist (minimum 4)
